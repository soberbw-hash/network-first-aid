# Changelog

## 0.1.1 - 2026-07-15

- 为全部 15 项修复工具新增悬浮与键盘聚焦说明，明确适用场景。
- 新增手动检查 GitHub Release 更新与安全下载引导。
- 将最小化按钮调整为辨识度更高的标准横线。
- 新增使用微信二维码的可选支持作者界面。
- 桌面运行时由 Electron 迁移到 Tauri 2 + Rust；安装版由约 107 MiB 缩小到约 18 MiB。
- 更新检查改为读取 Windows 系统代理并跟随官方 Release 跳转，不消耗 GitHub API 共享配额。
- 新增 TypeScript、Rust、Clippy、npm audit 与 RustSec 多层质量门禁。
- 新增单实例保护，重复启动时只唤醒已有窗口。

## 0.1.0 - 2026-07-15

- 首个公开版本。
- 新增 Windows 网络只读体检与代理/TUN/DNS/路由冲突分析。
- 新增 15 项固定白名单修复动作和风险分级。
- 新增修复前自动快照、选择性还原与本机审计记录。
- 新增安装版与免安装便携版。
- 完成 HarmonyOS Sans 与液态玻璃界面设计。
