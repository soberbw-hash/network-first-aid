## 不懂 Winsock，也能知道什么时候该点它

`v0.1.1` 把网络急救箱从“能修”继续往“敢让普通用户用”推进了一步。每个修复工具现在都有一个小问号，鼠标悬停或键盘聚焦后，会直接解释它适合什么情况。

### 这次更新

- 全部 15 项修复工具新增“什么时候用”场景说明；
- 标题栏新增手动检查更新，可识别最新 GitHub Release；
- 发现新版后才展示下载引导，跳转目标固定为本仓库 Release；
- 最小化按钮改为标准横线，与最大化按钮更容易区分；
- 新增不打扰主流程的“支持作者”入口和微信扫码界面；
- 底层从 Electron 迁移到 Tauri 2 + Rust，保留原有界面和本地快照数据；安装版由约 107 MiB 缩小到约 18 MiB；
- 更新检查会读取 Windows 当前系统代理，并通过 GitHub 官方 Release 跳转识别版本，不再消耗共享 API 配额；
- 现有 18 项 TypeScript 测试、6 项 Rust 单元测试与 2 项真实联网烟雾测试全部通过。

### 下载哪个？

- **Setup**：正常安装使用，本地快照与操作记录继续保存在原目录；
- **Portable**：免安装运行，适合先体验。

> 当前版本尚未进行商业代码签名，Windows SmartScreen 可能提示“未知发布者”。请只下载本 Release 附件，并使用 `SHA256SUMS.txt` 核对文件。
