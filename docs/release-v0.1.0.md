## 别再靠重启和反复开关代理碰运气

`v0.1.0` 是网络急救箱的首个公开版本。它不是又一个粗暴的“一键重置网络”，而是先回答：**谁接管了网络、超时发生在哪一层、修复后能不能安全回去。**

### 首发能力

- 一次体检覆盖系统代理、WinHTTP、DNS、默认路由、TUN、代理进程、监听端口、Hosts 与真实连通性；
- 分开验证本地代理端口和经代理访问 ChatGPT 的 CONNECT/TLS 出口；
- 识别多代理同时运行、失效 localhost 代理和断开的 TUN 默认路由；
- 提供 15 项固定白名单修复操作，并明确标注权限、风险和重启要求；
- 每个写操作前自动创建快照，代理、DNS、Hosts 与防火墙支持选择性还原；
- 所有网络数据和操作记录只保存在本机。

### 下载哪个？

- **Setup**：正常安装，支持桌面快捷方式；
- **Portable**：免安装直接运行，适合先体验。

> 当前版本尚未进行商业代码签名，Windows SmartScreen 可能提示“未知发布者”。请只下载本 Release 附件，并使用 `SHA256SUMS.txt` 核对文件。

如果它准确解释了你的网络为什么“明明有网却什么都很慢”，欢迎点一个 Star，并把复现环境提交到 Issues。
